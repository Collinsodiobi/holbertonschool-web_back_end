# Python - Async Comprehension

## Description
This project covers asynchronous generators, async comprehensions, and
type-annotating generators in Python.

## Tasks

### 0. Async Generator
File: `0-async_generator.py`

A coroutine `async_generator` that loops 10 times, asynchronously waits
1 second each time, and yields a random number between 0 and 10.
